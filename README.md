# 🚀 Aulas Programação Web

Aqui segue um repositório juntando meu aprendizado das aulas de **Programação Web Responsiva** do curso de Ciência da Computação.

## 💻 Sobre o Repositório
Este espaço contém o material de apoio, códigos-fonte e conceitos explorados durante as aulas, servindo como um registro da minha evolução técnica em:

*   **HTML5 & CSS3**: Estruturação semântica, estilização moderna e layouts responsivos com Flexbox.
*   **JavaScript (ES6+)**: Lógica de programação aplicada, manipulação do DOM, uso de Arrays e Objetos.
*   **Integração Frontend**: Conexão entre scripts e interfaces para criação de sistemas dinâmicos.

## 📚 Conteúdos Abrangidos
O repositório reflete o conhecimento adquirido em sala de aula, passando por fundamentos de design (UI/UX), lógica de gerenciamento de dados (CRUD) e boas práticas de desenvolvimento web.
