Aula 1

Resumo da aula: foi uma introdução ao desenvolvimento web, focada na tríade HTML, CSS e JavaScript.
Explicou a relação entre front-end, back-end e banco de dados, que o front solicita as informações, que o back puxa do banco e mostra pro front que mostra pro usuário. Além da importância de usar frameworks como Node.js para o mercado de trabalho. Na prática, configurámos o VS Code com extensões essenciais e criámos a estrutura padrão de pastas de um projeto. Aprendemos a anatomia das tags HTML (Head para SEO e Body para conteúdo) e como o CSS estiliza os elementos, e criamos uma função para um botão mostrar mensagem na tela.

Aula 3

A aula focou na base da Lógica de Programação com JavaScript, destacando a estrutura de variáveis (let), o uso de chaves para blocos de código e a importância do ponto e vírgula, conceitos que facilitam o aprendizado futuro de Java e C#. Você explorou a diferença entre linguagens fortemente tipadas (como C#) e o JS, praticou operadores aritméticos e de comparação (como o if/else, similar ao SE do Excel) e avançou na manipulação do DOM, conectando campos do HTML ao script para realizar funções de verificação e exibir resultados dinamicamente na página.
